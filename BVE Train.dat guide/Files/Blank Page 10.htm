#DEVICE

0 = Activates ATS-P.

0 = Activates ATC.

0 = Activates driver vigilance (deadman) device.

0 = Activates brake hold.

0 = Activates speed hold.

0 = Re-Adhesion device.

0 = Load compensation value.

0 = Activates error passing alarm.

 

0 = Denotes the device is inactive.

1 = Denotes the device is active unless otherwise listed below.

 

ATC:

1 = Manual switching mode,

2 = Automatic mode.

 

Re-Adhesion:

0 = Type A, use where wheelslip isn’t likely (LRV’s/Trams etc).

1 = Type B, use mainly on diesel multiple units (Don’t ask whyJ).

2 = Type C, use mainly on electric multiple units (ditto).

3 = Type D, use mainly on locos where wheelslip is expected.

 

Load Compensation:

Xx = Value in tons, adjusts acceleration/response depending on load.

 

Error passing alarm:

0 = Off

1 = Plays warning once (halt.wav for stop/passage.wav for not stop stations).

2 = Plays warning continuously.
