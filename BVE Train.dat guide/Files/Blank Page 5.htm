#BRAKE

0        = Brake type (see below).

1        = Regenerative/Rheostatic brake method (see below).

10      = Electric brake control speed (km/h).

 

Brake Types:

0        = Electro Pneumatic Brakes (self lapping),

1        = Electronically Controlled Pneumatic brakes (self lapping),

2        = Straight Air brake,

 

Regenerative/Rheostatic brake:

0        = Off

1        = Rheostatic, uses electric braking until control speed is reached then activates the service brake automatically,

2        = Regenerative, uses electric and service brakes until control speed is reached then uses service brake only.

 

NOTE: Regenerative/Rheostatic braking only works when the reverser is in forward (F) or backward (B).

 

Control speed:

The speed at which BVE automatically changes braking method, for straight air brakes this number should be higher than the trains maximum speed to avoid locking the wheels on braking.
